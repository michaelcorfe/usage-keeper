# Usage Keeper

> Track your Claude AI usage limits in real time, right from your browser toolbar.

![Version](https://img.shields.io/badge/version-1.0.0-green) ![License](https://img.shields.io/badge/license-MIT-blue)

## What it does

Never get caught off guard by Claude's usage limits again. Usage Keeper sits quietly in your browser toolbar and tracks your Claude session and weekly usage in real time. No more navigating to settings -- just click the icon and see exactly how much you have left and when it resets.

## Features

- Live session and weekly usage tracking
- Accurate reset timers for both session and weekly limits
- Color coded status -- green, yellow, orange, red at a glance
- Desktop notification when you hit 75% so you can plan ahead
- Auto refreshes every 5 minutes in the background
- No setup, no API keys -- just install and go

## Installation

1. Download the latest ZIP from the [Releases](https://github.com/SquareOneAI/usage-keeper/releases) page
2. Unzip the file
3. Go to `chrome://extensions` in your browser
4. Enable **Developer Mode** in the top right
5. Click **Load unpacked** and select the unzipped folder

> Once approved on the Chrome Web Store, installation will be one click.

## Requirements

- Chrome or Brave browser
- An active Claude.ai account (Free, Pro, or Max)
